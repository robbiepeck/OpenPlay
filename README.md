# OpenPlay

OpenPlay is a growing collection of open source, straightforward browser games built to stay free for the public to use forever.

The goal of this project is to create a long-term home for classic, easy-to-understand games that can be played directly in the browser, maintained in public, and expanded over time with clean, original implementations.

## Vision

OpenPlay is being built around a few simple ideas:

- Keep the games free and public.
- Focus on timeless, basic game concepts that are straightforward to build and easy to understand.
- Make the project expand cleanly so more games can be added over time.
- Prefer original implementations and presentation rather than copying third-party code or assets.

## Current Status

The first version of OpenPlay includes:

- A gallery-style homepage for the arcade portal
- A playable implementation of 2048
- Placeholder entries for future games including Minesweeper and Snake

## Roadmap

Planned next steps include:

- Add Minesweeper
- Add Snake
- Continue growing the library with more simple open source-friendly games
- Improve accessibility, polish, and cross-device playability

## Tech Stack

- React
- TypeScript
- Vite

## Local Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```
